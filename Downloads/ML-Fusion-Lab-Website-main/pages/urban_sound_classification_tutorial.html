Urban Sound Classification with Machine Learning

In this tutorial, we will build a machine learning model to classify urban sounds (e.g., car horns, sirens, drilling) using audio features and deep learning techniques. This kind of model can be used in smart cities, noise pollution monitoring, and automatic audio surveillance systems.

Prerequisites

  • Basic Python programming knowledge.
  • Familiarity with audio processing and machine learning.
  • Python environment (3.7+).
  • Required libraries installed: librosa, tensorflow, scikit-learn, matplotlib.

Step 1: Understanding Urban Sound Classification

Urban sound classification involves identifying different types of sounds commonly found in urban environments. Sounds like car horns, air conditioners, and street music are recorded as short clips and classified using machine learning models.

We’ll use deep learning techniques with audio feature extraction to build a sound classification model.

Step 3: Understanding Audio Features for Classification

Before building the model, we need to understand how to represent audio files. We’ll extract features like Mel-frequency cepstral coefficients (MFCCs), chroma, and spectral contrast from audio files.


import librosa
import numpy as np

# Load audio file
audio_path = 'sound.wav'
y, sr = librosa.load(audio_path, duration=5.0)

# Extract MFCCs
mfccs = librosa.feature.mfcc(y=y, sr=sr, n_mfcc=13)
print(mfccs.shape)

Step 5: Extracting Features for Model Input


def extract_features(file_name):
    try:
        y, sr = librosa.load(file_name, duration=5.0)
        mfccs = librosa.feature.mfcc(y=y, sr=sr, n_mfcc=40)
        return np.mean(mfccs.T, axis=0)
    except Exception as e:
        print(f"Error loading {file_name}: {e}")
        return None

Step 6: Building the Classification Model


from tensorflow.keras.models import Sequential
from tensorflow.keras.layers import Dense, Dropout

# Define the model
model = Sequential()
model.add(Dense(256, activation='relu', input_shape=(40,)))
model.add(Dropout(0.5))
model.add(Dense(128, activation='relu'))
model.add(Dropout(0.5))
model.add(Dense(10, activation='softmax'))  # 10 classes of urban sounds

model.compile(optimizer='adam', loss='sparse_categorical_crossentropy', metrics=['accuracy'])
