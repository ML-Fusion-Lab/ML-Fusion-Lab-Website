Pneumonia Detection using Machine Learning

Overview

This code implements a deep learning model for image classification using the FastAI library. It uses transfer learning with a ResNet50 architecture pre-trained on ImageNet. The code covers the complete machine learning pipeline from data loading to model training and evaluation.

Step 1: Imports and Setup


from fastai import *
from fastai.vision import *
from fastai.metrics import error_rate
import os
import pandas as pd
import numpy as np

What's happening here:

  • FastAI library imports for computer vision tasks
  • Pandas for data manipulation
  • NumPy for numerical operations

Step 2: Data Preparation


x  = 'Path'
path = Path(x)
path.ls()

np.random.seed(40)
data = ImageDataBunch.from_folder(path, 
    train = '.', 
    valid_pct=0.2,
    ds_tfms=get_transforms(), 
    size=224,
    num_workers=4).normalize(imagenet_stats)

What's happening here:

  • Setting up the data directory path
  • Creating an ImageDataBunch which:
    • Splits data into 80% training, 20% validation
    • Applies standard image transformations
    • Resizes images to 224x224 pixels
    • Normalizes using ImageNet statistics

Step 3: Model Creation and Initial Training


learn = cnn_learner(data, 
    models.resnet50, 
    metrics=[accuracy], 
    model_dir = Path('Path'),
    path = Path("."))

learn.lr_find()
learn.recorder.plot(suggestions=True)

lr1, lr2 = 1e-3, 1e-1
learn.fit_one_cycle(4, slice(lr1,lr2))

What's happening here:

  • Creates a CNN learner using ResNet50 architecture
  • Finds optimal learning rate using the LR Finder
  • Trains for 4 epochs using the One Cycle Policy
  • Learning rate varies from 0.001 to 0.1

Step 4: Fine-tuning


learn.fit_one_cycle(20,slice(lr))
learn.unfreeze()
learn.lr_find()
learn.recorder.plot()
learn.fit_one_cycle(10,slice(1e-4,1e-3))

What's happening here:

  • Additional training for 20 epochs
  • Unfreezes all layers for fine-tuning
  • Finds new optimal learning rate
  • Final training for 10 epochs with smaller learning rates

Step 5: Model Evaluation


learn.recorder.plot_losses()
interp = ClassificationInterpretation.from_learner(learn)
interp.plot_confusion_matrix()

What's happening here:

  • Plots the training losses
  • Creates interpretation object
  • Visualizes model performance using confusion matrix

Step 6: Making Predictions


img = open_image('IM-0001-0001.jpeg')
print(learn.predict(img)[0])

What's happening here:

  • Loads a test image
  • Makes prediction using trained model
  • Prints predicted class

Important Notes

  • Ensure your image directory structure follows FastAI's requirements
  • Adjust learning rates based on your specific dataset
  • Monitor validation metrics to prevent overfitting
  • Consider using GPU for faster training
