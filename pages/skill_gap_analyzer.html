Skill Gap Analyzer: Using Flask, Proxycurl, and Gemini APIs

1. Project Overview

This project builds a Flask web application that helps users identify skill gaps based on their career goals and current skills. The app uses the Proxycurl API to extract skills from LinkedIn profiles, and the Gemini API (from Google) to suggest additional skills to acquire.

2. Setting Up the Environment

To begin, you need to install the required libraries and packages:

pip install Flask google-generativeai requests
pip install gunicorn # For deployment
pip install python-dotenv # To manage environment variables

3. Application Architecture

The app consists of two key components:

  • Fetching LinkedIn skills using Proxycurl API
  • Using Google Gemini API to suggest skill gaps based on the user's career goals

4. Defining API Functions

4.1 Fetching LinkedIn Profile Data

First, let's define the function that uses the Proxycurl API to fetch LinkedIn profile details and skills:

import requests PROXYCURL_API_KEY = 'YOUR_PROXYCURL_API_KEY' PROXYCURL_API_ENDPOINT = 'https://nubela.co/proxycurl/api/v2/linkedin' def get_linkedin_profile(linkedin_url): """Fetches the LinkedIn profile using Proxycurl API.""" headers = { 'Authorization': f'Bearer {PROXYCURL_API_KEY}', } params = { 'linkedin_profile_url': linkedin_url, 'extra': 'include', 'skills': 'include', 'use_cache': 'if-present', 'fallback_to_cache': 'on-error', } try: response = requests.get(PROXYCURL_API_ENDPOINT, params=params, headers=headers) if response.status_code == 200: return response.json() # Return profile data else: return {"error": f"Error fetching profile, status code: {response.status_code}"} except Exception as e: return {"error": f"Error during LinkedIn API call: {e}"}

4.2 Gemini API for Skill Gap Analysis

Next, define the function that interacts with Google Gemini API to identify skill gaps:

import google.generativeai as genai import os def get_gemini_response(career_goal, skills): """Fetches response from Gemini API based on career goal and skills.""" query = f"Considering my career goal of '{career_goal}', what additional skills would I need to acquire if my current skills are {', '.join(skills)}? Just list them concisely." model = genai.GenerativeModel('gemini-pro') api_key = os.getenv("GOOGLE_API_KEY") # Load API key from environment variable genai.configure(api_key=api_key) try: response = model.generate_content(query) return response.text except Exception as e: return f"Error occurred during Gemini API call: {e}"

5. Setting Up the Flask App

5.1 Flask Routes

The app uses a basic form to collect user input (career goals and skills), and then fetches LinkedIn data or user-provided skills. Here is the Flask code:

from flask import Flask, request, render_template import json app = Flask(__name__) @app.route('/', methods=['GET', 'POST']) def index(): if request.method == 'POST': career_goal = request.form['careerGoal'] manual_skills = request.form.getlist('skill[]') linkedin_url = request.form.get('linkedinProfileUrl') profile_data = None if linkedin_url: profile_data = get_linkedin_profile(linkedin_url) if 'error' in profile_data: return render_template('index.html', error=profile_data['error']) else: profile_data = manual_skills if profile_data: gemini_response = get_gemini_response(career_goal, profile_data['skills'] if 'skills' in profile_data else manual_skills) return render_template('index.html', profile_data=profile_data, gemini_response=gemini_response) else: return render_template('index.html', error="Please enter your career goal and skills.") return render_template('index.html') if __name__ == '__main__': app.run(debug=True, use_reloader=False)

6. Testing the App

Run the Flask application locally by executing the following command in your terminal:

flask run

Visit http://localhost:5000 to interact with the app. You can input your career goal, LinkedIn URL, or skills to get recommendations.

7. Conclusion

This Flask app integrates with the Proxycurl API to fetch LinkedIn profile data and the Gemini API to recommend additional skills based on user-provided career goals. This tool can be expanded to include further career planning features, making it a powerful career guidance application.
