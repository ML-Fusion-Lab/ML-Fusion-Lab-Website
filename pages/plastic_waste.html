Plastic Waste Classification To Recycle

1. Import Libraries and Data Preparation


    import pandas as pd
    import numpy as np
    import tensorflow as tf
    from sklearn.model_selection import train_test_split
    from sklearn.preprocessing import LabelEncoder
    from tensorflow.keras.preprocessing.image import ImageDataGenerator
    import matplotlib.pyplot as plt
    import seaborn as sns
    import warnings
    
    warnings.filterwarnings("ignore")
    
    # Load dataset containing images of plastic waste and their labels
    df = pd.read_csv("plastic_waste_data.csv")
    df.head()

This section initializes libraries for data manipulation (pandas, numpy), machine learning (TensorFlow), and data visualization (matplotlib, seaborn). The dataset is loaded, containing images of plastic waste along with their corresponding labels for classification.

2. Data Visualization


    plt.figure(figsize=(10, 6))
    sns.countplot(data=df, x='plastic_type', palette='Set2')
    plt.title('Distribution of Plastic Types')
    plt.xlabel('Plastic Type')
    plt.ylabel('Count')
    plt.show()

This code visualizes the distribution of different types of plastic waste in the dataset, helping to identify which categories are most prevalent and may require more focused recycling efforts.

3. Data Preprocessing and Augmentation


    # Encode labels
    label_encoder = LabelEncoder()
    df['encoded_labels'] = label_encoder.fit_transform(df['plastic_type'])
    
    # Set up data augmentation for image data
    data_gen = ImageDataGenerator(
        rescale=1.0/255,
        rotation_range=20,
        width_shift_range=0.2,
        height_shift_range=0.2,
        shear_range=0.2,
        zoom_range=0.2,
        horizontal_flip=True,
        fill_mode='nearest'
    )
    
    # Split dataset into training and validation sets
    train_data, val_data = train_test_split(df, test_size=0.2, random_state=42)

Prepare the data by encoding labels and setting up data augmentation to enhance the training dataset. The dataset is then split into training and validation sets.

4. Convolutional Neural Network for Plastic Classification


    from tensorflow.keras.models import Sequential
    from tensorflow.keras.layers import Conv2D, MaxPooling2D, Flatten, Dense, Dropout
    
    # Build a convolutional neural network model
    model = Sequential([
        Conv2D(32, (3, 3), activation='relu', input_shape=(150, 150, 3)),
        MaxPooling2D(pool_size=(2, 2)),
        Conv2D(64, (3, 3), activation='relu'),
        MaxPooling2D(pool_size=(2, 2)),
        Conv2D(128, (3, 3), activation='relu'),
        MaxPooling2D(pool_size=(2, 2)),
        Flatten(),
        Dense(128, activation='relu'),
        Dropout(0.5),
        Dense(len(df['plastic_type'].unique()), activation='softmax')  # Output layer for classification
    ])
    
    model.compile(optimizer='adam', loss='sparse_categorical_crossentropy', metrics=['accuracy'])
    model.fit(data_gen.flow(train_data['image_path'], train_data['encoded_labels'], batch_size=32), epochs=30)

Train a convolutional neural network (CNN) model to classify plastic waste types based on images, facilitating the sorting and recycling process.

5. Evaluation of Plastic Classification Model


    # Evaluate model performance on validation set
    val_loss, val_accuracy = model.evaluate(data_gen.flow(val_data['image_path'], val_data['encoded_labels'], batch_size=32))
    print(f"Validation Loss: {val_loss:.2f}")
    print(f"Validation Accuracy: {val_accuracy:.2f}")
    
    # Visualize training history
    plt.plot(model.history.history['accuracy'], label='Training Accuracy')
    plt.plot(model.history.history['val_accuracy'], label='Validation Accuracy')
    plt.title('Model Accuracy')
    plt.xlabel('Epoch')
    plt.ylabel('Accuracy')
    plt.legend()
    plt.show()

Evaluate the performance of the plastic classification model using validation loss and accuracy. Visualize the training and validation accuracy over epochs to assess model training quality.

6. Plastic Waste Classification and Recycling


    # Load new sample image for classification
    from tensorflow.keras.preprocessing import image
    
    img = image.load_img('new_plastic_sample.jpg', target_size=(150, 150))
    img_array = image.img_to_array(img)
    img_array = np.expand_dims(img_array, axis=0) / 255.0
    
    # Predict the type of plastic waste
    predicted_class = model.predict(img_array)
    predicted_label = label_encoder.inverse_transform([np.argmax(predicted_class)])
    print(f"Identified Plastic Type: {predicted_label[0]}")

Implement the trained model to classify new images of plastic waste, identifying the type and enabling appropriate recycling measures.

7. Conclusion

The Plastic Waste Classification system utilizes deep learning techniques to accurately identify different types of plastic waste from images. This classification helps streamline the recycling process, promoting more effective waste management and environmental sustainability.
